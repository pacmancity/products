# React + TypeScript + Vite

#### установка

```bash
npm i
```

#### запуск

```bash
npm start
```

Открыть в браузере: http://localhost:3000/

#### сборка

```bash
npm run build
```

Проект собирается в папке build

#### запуск билда локально

```bash
npm run preview
```

Открыть в браузере: http://localhost:4173/
